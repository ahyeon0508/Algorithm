# x만큼 간격이 있는 n개의 숫자

-------
### 🌞 문제
함수 solution은 정수 x와 자연수 n을 입력 받아, x부터 시작해 x씩 증가하는 숫자를 n개 지니는 리스트를 리턴해야 합니다. 다음 제한 조건을 보고, 조건을 만족하는 함수, solution을 완성해주세요.

<b>제한사항</b>  
- x는 -10000000 이상, 10000000 이하인 정수입니다.
- n은 1000 이하인 자연수입니다.

### 📝 입출력
|x|n|answer|
|---|---|---|
|2|5|[2,4,6,8,10]|
|4|3|[4,8,12]|
|-4|2|[-4, -8]|

### 👩‍💻 풀이
```python
def solution(x, n):
    answer = [x * i for i in range(1, n+1)]
    return answer
 ```

### 🔑 구현 아이디어
- for문을 이용하면 된다.
