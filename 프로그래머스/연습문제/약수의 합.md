# 약수의 합

-------
### 🌞 문제
정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.

<b>제한사항</b>  
- n은 0 이상 3000이하인 정수입니다.

### 📝 입출력
|n|return|
|---|---|
|12|28|
|5|6|

### 👩‍💻 풀이
```python
def solution(n):
    answer = 0
    for i in range(1, n+1):
        if n % i == 0:
            answer += i
    return answer
 ```

### 🔑 구현 아이디어
- 약수의 성질을 for문을 통해 구현하면 된다.
  
### 🙋‍♀ 문제에 대한 나의 생각
- n / 2를 이용해서 성능을 약 2배 향상 시킨 분들을 보고 와...라는 생각만 들었다.
